# 🚀 Smart Email AI Assistant

An AI-powered system that classifies emails and generates smart replies to improve productivity.

---

## 📌 Features

* 📩 **Email Classification**

  * Detects Spam, Important, Urgent, or Normal emails

* 💬 **Smart Reply Generator**

  * Suggests quick replies based on email content

* ⚡ **Confidence Score**

  * Shows how confident the AI is (High / Medium)

* 🧠 **Email Summary**

  * Generates a short summary of long emails

* 🎨 **Interactive UI**

  * Built using Gradio for a clean and simple interface

---

## 🛠️ Tech Stack

* Python 🐍
* Gradio 🎨
* Hugging Face Spaces 🤗

## ⚙️ How It Works

1. User enters an email
2. AI analyzes keywords
3. Classifies email into categories
4. Generates a smart reply
5. Displays confidence score and summary
